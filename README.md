BMI Calculator

This project is developed as part of the **Oasis Infobyte (AICTE OIBSIP) Python Programming Internship**.

 Project Description
The BMI (Body Mass Index) Calculator takes user input for height and weight, calculates BMI, and categorizes the result into health ranges such as Underweight, Normal, Overweight, or Obese.

 Features
- Beginner version (Command Line Interface)
- Advanced version (GUI using Tkinter)
- Input validation
- Health category classification
- User-friendly interface (GUI)

 Technologies Used
- Python
- Tkinter (for Advanced version)

 How to Run
```bash
python bmi_beginner.py
